Estruturas de Seleção em Python

O que são Estruturas de Seleção?

Estruturas de seleção são comandos que permitem ao programa tomar decisões e executar diferentes blocos de código com base em condições. Elas são fundamentais para criar programas que respondem de maneira diferente a diferentes situações.

Em Python, as principais estruturas de seleção são:

  • if: Executa um bloco de código se uma condição for verdadeira
  • if-else: Executa um bloco se a condição for verdadeira, e outro bloco se for falsa
  • if-elif-else: Permite verificar múltiplas condições em sequência

Aplicações

As estruturas de seleção são usadas em praticamente todos os programas, pois permitem:

  • Validar entradas de usuários
  • Implementar lógica de negócios com diferentes caminhos
  • Tratar casos especiais e exceções
  • Criar menus e interfaces interativas
  • Controlar o fluxo de execução do programa

Exemplos de Código

Estrutura if simples


# Verificando se um número é positivo
numero = 10

if numero > 0:
    print("O número é positivo")

Estrutura if-else


# Verificando se um número é par ou ímpar
numero = 7

if numero % 2 == 0:
    print("O número é par")
else:
    print("O número é ímpar")

Estrutura if-elif-else


# Classificando notas
nota = 85

if nota >= 90:
    print("Conceito A")
elif nota >= 80:
    print("Conceito B")
elif nota >= 70:
    print("Conceito C")
elif nota >= 60:
    print("Conceito D")
else:
    print("Conceito F")
