Estruturas de Repetição em Python

O que são Estruturas de Repetição?

Estruturas de repetição, também conhecidas como loops, são comandos que permitem executar um bloco de código múltiplas vezes. Elas são essenciais para automatizar tarefas repetitivas e processar coleções de dados.

Em Python, as principais estruturas de repetição são:

  • for: Usado para iterar sobre uma sequência (como lista, tupla, dicionário, conjunto ou string)
  • while: Executa um bloco de código enquanto uma condição for verdadeira

Aplicações

As estruturas de repetição são fundamentais para:

  • Processar listas, arrays e coleções de dados
  • Realizar operações em cada elemento de uma sequência
  • Executar código até que uma condição específica seja atingida
  • Implementar algoritmos iterativos
  • Automatizar tarefas repetitivas

Exemplos de Código

Loop for com lista


# Iterando sobre uma lista
frutas = ["maçã", "banana", "laranja", "uva"]

for fruta in frutas:
    print(f"Eu gosto de {fruta}")

Loop for com range()


# Usando range() para gerar uma sequência de números
for i in range(5):  # Gera números de 0 a 4
    print(f"Número: {i}")

# Range com início, fim e passo
for i in range(2, 10, 2):  # Números pares de 2 a 8
    print(f"Número par: {i}")

Loop while


# Contagem regressiva usando while
contador = 5

while contador > 0:
    print(f"Contagem: {contador}")
    contador -= 1  # Decrementa o contador
print("Fim da contagem!")

Controle de loops com break e continue


# Usando break para sair do loop
for i in range(10):
    if i == 5:
        print("Encontrei o 5! Saindo do loop...")
        break
    print(f"Número: {i}")

# Usando continue para pular uma iteração
for i in range(10):
    if i % 2 == 0:  # Se o número for par
        continue    # Pula para a próxima iteração
    print(f"Número ímpar: {i}")

Loop for com enumerate()


# Usando enumerate() para obter índice e valor
frutas = ["maçã", "banana", "laranja", "uva"]

for indice, fruta in enumerate(frutas):
    print(f"Índice {indice}: {fruta}")
